Ordenação - Conceito

Ordenação, em inglês sort é uma das tarefas com maior impacto na eficiência de soluções computationais. Sejam elas baseadas em software ou em hardware.

Problema: dado um conjunto de itens quaisquer, de que forma podemos organizar estes itens segundo uma sequência especificada.

Ordenação - Conceito

Por ser tão importante os problemas de ordenção são muito estudados e temos, centenas de algoritmos diferentes para resolver problemas deste tipo.

A escolha do algoritmo de ordenação dependerá do conjunto de itens que será ordenado, da infraestrutura disponível, da ordem inicial dos itens e dos custos envolvidos. Nem sempre o algoritmo mais rápido será o mais eficaz para um dado problema.

Ordenação - Conceito Exemplo

Ordenação - Conceito

Raramente você irá ordenar uma lista de números, ou palavras, ou arquivos. Profissionalmente, a maior parte dos problemas de ordenamento são relacionados a dados complexos records compostos de diversos campos de tipos diferentes com um ou mais campos chave keys que deverão ser usados como referência.

Definição do Problema

Uma coleção de elementos $A$ cujos elementos são indicados por $a_i$ ou $A[i]$. O objetivo de uma ordenação crescente será criar um conjunto $A$ ordenado de tal forma que para todos os elementos $A[i]$ exista um $A[j]$ desde que $A[i] < A[j]$ onde $j > i$ e não exista nenhum elemento $A[k]$ entre $A[i] \wedge A[j]$.

Definição do Problema

Para ordenar os elementos usaremos uma permutação comparando, de alguma forma, os elementos da coleção $A$ para definir se um $A[i]$ qualquer é maior, igual ou menor que um $A[i]$ qualquer. A forma como fazemos esta comparação e a permutação é que define como os algoritmos de ordenação funcionam.

Bubble Sort

O Bubble Sort é um dos mais simples algoritmos de ordenação (IVERSON, 1962) a ideia é percorrer toda a coleção de dados $A$ comparando $A[j]$ com $A[j+1]$ e inverter a posição destes dois elementos sempre que $A[j] > A[j+1] $. 


                1. begin
                2.    for i = 0 to n 
                3.     for j = i to n − i
                4.       if (A[j] > A[j + 1]) then
                5.         Swap (A[j], A[j + 1])
                8. end
